# GameZone - Loja Gamer

## 📌 Descrição
GameZone é uma loja online de periféricos e acessórios gamer desenvolvida com HTML5, CSS3 e JavaScript puro. O projeto conta com um carrinho de compras funcional, navegação responsiva e páginas para portfólio, sobre e contato.

---

## 🎯 Funcionalidades

- ✅ **Home**: Página inicial com promoções em destaque
- ✅ **Portfólio**: Catálogo de produtos com opção de adicionar ao carrinho
- ✅ **Carrinho de Compras**:
  - Adicionar/remover produtos
  - Incrementar quantidade de itens
  - Cálculo automático de total
  - Finalizar compra
- ✅ **Página de Contato**: Formulário de mensagens
- ✅ **Página Sobre**: Informações sobre a GameZone

## 🚀 Como Usar

### Pré-requisitos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)

### Instalação
1. Clone o repositório:
```bash
git clone https://github.com/fishpedrol/cp03-front-end-fiap.git
```

2. Abra o arquivo `index.html` no navegador ou use um servidor local

### 🌐 GitHub Pages

Acesse o projeto online diretamente pelo GitHub Pages:
👉 [https://fishpedrol.github.io/cp03-front-end-fiap/](https://fishpedrol.github.io/cp03-front-end-fiap/)

Para ativar o GitHub Pages no seu repositório:
1. Vá em **Settings** → **Pages**
2. Selecione **Deploy from a branch**
3. Escolha a branch **main** (ou **master**) e a pasta **/(root)**
4. Clique em **Save**
5. Aguarde alguns minutos e acesse o link fornecido

## ⚙️ Funcionalidades do Carrinho

### JavaScript
- `adicionarAoCarrinho()`: Adiciona produto ao carrinho ou incrementa quantidade
- `removerDoCarrinho()`: Remove item do carrinho
- `atualizarCarrinho()`: Atualiza display e totais
- `abrirCarrinho()`: Abre painel do carrinho
- `fecharCarrinho()`: Fecha painel do carrinho
